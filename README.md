# Python Basic Arithmetic Operations & Greeting Program

## Task 1: Basic Arithmetic Operations
This program takes two numbers as input from the user and performs:
- Addition
- Subtraction
- Multiplication
- Division

The result of each operation is displayed on the screen.

## Task 2: Personalized Greeting
This program takes the user's first name and last name as input, combines them into a full name, and prints a personalized greeting message.

## Language Used
Python 3
